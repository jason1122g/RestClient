# RestClient
An application run in Firefox OS that helps web developers to test custom HTTP requests

![PIC](http://g.recordit.co/aOHesoNVes.gif)

## Feature

- When Request
  - Custom Method
  - Custom Query Parameter
  - Custom Header
  - Custom Body
    - Formatted in JSON
    - Formatted in Form Encoded


- When Response
  - Show Status
  - Show Body
    - Formatted JSON
    - Formatted HTML
    - Plain Text
